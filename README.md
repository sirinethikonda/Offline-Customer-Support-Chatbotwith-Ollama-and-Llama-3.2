# Offline Customer Support Chatbot

This project demonstrates how to build a private, offline customer support chatbot using **Ollama** and the **Llama 3.2 3B** model.

## Key Features
- **Data Privacy**: All processing happens locally on your machine.
- **Cost Effective**: No API costs for model inference.
- **Prompt Engineering**: Comparison between zero-shot and one-shot prompting techniques.
- **Manual Evaluation**: A structured rubric to assess model performance.

## Project Structure
- `prompts/`: Prompt templates for the LLM.
- `eval/`: Evaluation results and scoring.
- `chatbot.py`: Python script for interacting with the Ollama API.
- `report.md`: Analysis of the prompting strategies.
